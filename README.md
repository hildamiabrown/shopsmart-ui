# ShopSmart UI - Responsive E-Commerce Interface

GitHub repository: [github.com/hildabrown/shopsmart-ui](https://github.com/hildabrown/shopsmart-ui)

ShopSmart UI is a responsive front-end e-commerce interface built with HTML, CSS, and JavaScript. It includes dynamic product rendering, live search, category filtering, mobile navigation, cart feedback, and responsive layouts optimized for desktop and mobile screens.

## Files

- `index.html` - page structure and semantic content
- `styles.css` - responsive layout, visual design, and component styling
- `script.js` - product data, DOM rendering, filters, cart interactions, and mobile menu behavior

## Features

- Responsive e-commerce landing and product grid interface
- JavaScript-powered product cards rendered from structured data
- Live product search and category filtering
- Interactive cart count and toast notifications
- Mobile-friendly navigation and flexible layouts
- Cross-browser-friendly HTML, CSS, and JavaScript

## How to Run

Open `index.html` in a browser, or deploy the project with GitHub Pages from the repository root.
